# Itomori's Cool CLI

_Itomori's CLI is cool and helpful!_

**Some helpful commands :**

***1 : `-h` or `--help`:***

Run - `Itomori -h` or `Itomori --help` to see a brief info's about all commands.


***2 : `--version`:***

Run - `Itomori --version` to see a current version of Itomori, you are using!.

***3 : `--update`:***

Run - `Itomori --update` to see update Itomori automatically with `uv`.

***4 : `--license`:***

Run - `Itomori --license` to see the short license preview.

***5 : `--fullLicense`:***

Run - `Itomori --fullLicense` to see the full license text.

***6 : `--uninstall`:***

Run - `Itomori --version` to uninstall Itomori.
