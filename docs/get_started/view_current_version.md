# View Itomori's version

To view Itomori's current version from CLI is easy

**Follow these steps (CLI Way) :**
1. Open your terminal or command prompt
2. Run this command to view current Itomori version :-
    ```
        Itomori --version
    ```
3. Now it should provide the current Itomori version

**Done !**


_Or, if you prefer to view the Itomori's current version from the App_

**Follow these steps (TUI way):**

1.  Open Itomori App
2. Press `v` key from your keyboard , now you should see the about page, which include's all the About info , included version

**Done!**
